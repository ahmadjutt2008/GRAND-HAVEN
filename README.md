# Grand Haven | Luxury Hotel & Resort

A premium, responsive hotel booking website template designed to showcase luxury accommodations and provide a seamless user experience.

**Grand Haven** is a static web application built to represent a high-end hotel and resort. It features a modern, elegant design with smooth animations, a responsive layout, and a user-friendly interface. The site includes sections for hotel listings, room suites, amenities, special offers, guest testimonials, and a photo gallery.
